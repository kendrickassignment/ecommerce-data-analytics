Proyek ini merupakan analisis data dari dataset E-Commerce Public Dataset. Dashboard ini dibuat untuk memberikan gambaran mengenai performa penjualan, demografi pelanggan, dan analisis RFM (Recency, Frequency, Monetary).

## Struktur Direktori
- **/dashboard**: Berisi file utama untuk menjalankan dashboard Streamlit (`dashboard.py`) dan dataset yang telah dibersihkan (`main_data.csv`).
- **/data**: Berisi dataset mentah dalam format CSV.
- **notebook.ipynb**: File Jupyter Notebook yang berisi proses analisis data mulai dari Gathering, Assessing, Cleaning, hingga EDA.
- **requirements.txt**: Daftar library Python yang dibutuhkan untuk menjalankan proyek.
- **url.txt**: Tautan dashboard yang telah dideploy (jika ada).
